# Salary Prediction using Machine Learning

## Objective

Predict salary based on experience and other relevant features.

## Dataset

Dataset containing experience and salary information.

## Tools Used

* Python (Pandas, NumPy)
* Visualization (Matplotlib)
* Machine Learning (Scikit-learn)

## Analysis Performed

* Data preprocessing
* Relationship analysis between experience and salary

## Models Used

* Linear Regression

## Results

* Model shows strong relationship between experience and salary

## Key Insights

* Salary increases with experience
* Linear trend observed

## Future Work

* Add more features for better prediction accuracy
